# Server RPG System

## Overview
This plugin serves as the core progression and leveling system for the entire server. 
Built upon the powerful AuraSkills framework, it has been heavily customized to introduce new skills, unique ability systems, and exclusive crafting mechanics tailored to our server's gameplay loop.

## Core Features
* **AuraSkills Foundation:** Reliable, highly optimized, and balanced core progression.
* **Custom Skill Systems:** New custom skills and modified vanilla categories to fit the server's theme.
* **Personalized Abilities:** Unique perks, passive buffs, and active magic powers unlocked as players level up.
* **Unique Crafting:** Custom rune creation, exclusive block crafting, and specialized armor smithing.

## Skills Framework

### Default Core Skills
* Farming, Foraging, Mining, Fishing, Excavation, Archery, Defense, Fighting, Agility, Enchanting, Alchemy.

### Merged & Removed Skills
To streamline the experience, some default AuraSkills have been adjusted:
* **Endurance** -> Merged into *Agility*
* **Healing** -> Merged into *Alchemy*
* **Forging** -> Merged into *Enchanting*
* **Sorcery** -> Removed from the default pool

### Custom Additions & Expansions
* **Social:** Server-wide interactions and party-based benefits.
* **Herbalism:** Ability to harvest specific plants and crops (excluding trees and farming blocks).
* **Cooking:** Increases food regeneration, multiplies output, and grants special buffs.
* **Endurance & Magic Endurance:** Scales physical and magical defense attributes.
* **Magic:** Grants players the ability to unlock and cast special powers.
* **RuneCrafting:** A custom system to craft magical runes.
* **Engineering:** Specialized progression for redstone mechanics and automation.
* **Crafting & Smithing:** Unlocks the creation of custom blocks and specialized armor.
* **World & Combat Skills:** Swimming, Hunter, Slayer, and Taming.

## Installation & Configuration
1. Drop the compiled `.jar` file into your server's `plugins` folder.
2. Restart the server to generate the configuration files.
3. Navigate to the plugin's folder to tweak EXP curves, rewards, and custom recipes on `.yml` fiels.
